# 🏋️‍♂️ ATLAS-FITNESS

## 📌 Overview
ATLAS FITNESS is a desktop management application designed for gyms and fitness centers.  
It helps manage members, subscriptions, sessions, and payments through a modern and user-friendly interface.

Developed using **C# (Windows Forms)** and **SQL Server**, the system ensures performance, security, and reliability.

---

## 🚀 Features

- 👤 Member Management (Add / Edit / Delete)
- 📅 Subscription & Renewal Management
- 🕒 Session Scheduling
- 💳 Payment Tracking
- 📊 Dashboard & Statistics
- 📁 Excel Export
- 📷 QR Code Generation
- 🔐 User Authentication & Role Management

---

## 🛠️ Technologies Used

- **Frontend:** C# – Windows Forms  
- **Backend:** .NET Framework  
- **Database:** SQL Server  
- **Data Access:** ADO.NET  
- **Excel Export:** ClosedXML  

---

## 🗄️ Database Main Tables

- Persons  
- Clients  
- Subscriptions  
- Payments  
- Sessions  
- Users
